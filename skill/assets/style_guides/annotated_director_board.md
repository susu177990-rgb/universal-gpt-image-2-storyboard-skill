# Annotated Director Storyboard Sheet Style Guide

This style guide defines the prompt structure required to generate a complete, unified "Filmmaking Pitch & Storyboard Sheet" (提案级导演分镜纸) in a single image.

## 6-Section Layout Structure
The generated sheet must contain the following 6 distinct visual zones:

1. **Header Row (项目头栏):**
   - Project metadata, runtime, and aspect ratio.

2. **Character & Styling Lock Row (定妆锁定栏):**
   - Close-up reference frames labeled: "CHARACTER" (multi-angle facial views), "OUTFIT/COSTUME" (garment textures), "HAIR" (hair profile), and "ACCESSORIES/PROPS" (key gear details).

3. **Camera & Movement Plan Zone (相机与运动区):**
   - Visual mockups of camera bodies, lens types (e.g., "24mm", "50mm"), and graphic flight path diagrams (e.g., aerial push-in, crane pan) labeled with directional arrows.

4. **Expressions & Action Keyframes Zone (表情与动作区):**
   - A grid of small character face and pose close-ups (e.g., 2x4 grid) showing emotional keyframes (e.g., focused, breathing, scanning).

5. **Core Storyboard Grid (分镜区):**
   - The main rectangular panels showing sequence actions with bold "PANEL" numbers and clear caption boxes below for "ACTION" and "DIALOGUE".

6. **Technical Specs Footer (技术规格栏):**
   - Rows for: "LIGHTING CONTINUITY" (lighting setups), "MATERIALS & VFX" (textures like ripstop, water drops, metal details), and "COLOR SCRIPT" (color palette frames).
