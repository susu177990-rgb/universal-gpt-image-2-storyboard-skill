# Cinematic Storyboard Language Guide

Use this unified language to ensure prompts are short, precise, and actionable. Avoid vague adjectives.

## Language Modules

### 1. Shot Size
- Extreme Wide Shot (EWS)
- Wide Shot (WS)
- Full Shot (FS)
- Medium Shot (MS)
- Close-up (CU)
- Extreme Close-up (ECU)

### 2. Camera Position / Angle
- Eye-level
- Low angle (Worm's eye)
- High angle (Bird's eye)
- Dutch angle (canted)
- Over-the-shoulder (OTS)
- Point of View (POV)

### 3. Composition
- Rule of thirds
- Center framed
- Symmetrical composition
- Leading lines
- Foreground framing (dirty frame)

### 4. Character Position
- Frame left / Frame right
- Center
- Foreground / Midground / Background

### 5. Spatial Anchor
- "Relative to the doorway"
- "Beside the main window"
- "In the center of the room"

### 6. Action Phase
- Anticipation (wind-up)
- Mid-action (in motion)
- Impact / Peak
- Follow-through / Recovery

### 7. Performance Focus
- Subtle eye dart
- Clenched jaw
- Relaxed posture
- Wide-eyed shock

### 8. Shot Function
- Establish geography
- Reveal detail
- Show reaction
- Connect characters
