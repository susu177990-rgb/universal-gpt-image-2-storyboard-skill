# 长镜头分镜板蓝图 (统一六区提案表)

*此蓝图将“六区统一分镜”布局应用于描绘连续动作的单一、不间断镜头 (一镜到底)。它要求在所有面板中保持空间、照明和角色的连续性。*

## 区域 1：头部信息栏 (元数据)
- **标题/场景**: 一镜到底/连续序列 [插入场景，例如：“走廊边走边谈”]
- **序列类型**: 长镜头 (跟踪 / 斯坦尼康)
- **帧率 / 运行时长**: 基础 24fps / 目标 10-20 秒
- **氛围/基调**: 流畅、沉浸式、不间断的张力

## 区域 2：角色与造型锁定栏
- **主体**: 确认所有移动的主体。确立他们的相对身高和步调。
- **服装动态**: 确保服装的背面和侧面轮廓被清晰定义，因为摄影机很可能会围绕他们旋转。
- **道具锁定**: 整个镜头中携带的物品。

## 区域 3：摄影机与运动规划区
- **机身/镜头**: 斯坦尼康 (Steadicam) 或稳定器 (Gimbal) 套件。广角到中焦镜头（例如 24mm 到 50mm）以允许空间跟踪而不会丢失主体。
- **摄影机运动轨迹**: 
  - 需要绝对的连续性。 
  - 规划出精确的轨迹：`从角色 A 的脸部开始 -> 拉回镜头露出角色 B -> 沿着走廊向后跟踪拍摄 -> 甩镜头 (Whip pan) 对准门口。`

## 区域 4：表情与动作关键帧区
- 规划在连续移动*期间*发生的微交互：
  1. *递交*: 在角色之间平滑传递的道具。
  2. *越肩回眸*: 主体进行的快速空间检查。
  3. *环境交互*: 行走中途手拂过墙壁或推开门。

## 区域 5：核心分镜网格 (顺序面板)
*(侧重于连续的流动性、空间一致性和符合逻辑的视角转换。使用所需数量的面板。)*

- **面板 1**: 起点 (Start State)
  - *动作*: 确立初始位置和摄影机锚点。
- **面板 2**: 发展 (Development / Movement)
  - *动作*: 展示摄影机推进或角色移动。背景平滑改变。
- **面板 3**: 中点交汇 (Mid-Point Pivot / Action)
  - *动作*: 摄影机旋转或主体转身。保持相对于房间的确切照明方向。
- **面板 4**: 高点 (Peak Action)
  - *动作*: 突出连续运动的高潮。最佳构图。
- **面板 5**: 停点 (Stop / Resolution)
  - *动作*: 明显的减速或静止状态。提供一个稳定的结束画面。

## 区域 6：技术规格页脚
- **照明**: 锁定环境照明。当摄影机移动时，角色穿过预先布光的区域（例如，经过实际的头顶灯下，从阴影移动到窗户光线下）。
- **材质/视觉特效**: 无缝过渡，没有跳剪。移动穿过画面的实际元素以产生视差 (parallax)。
- **色彩脚本**: 渐进式的转变（例如，从暖色调的房间进入冷色调的走廊）。
