# 导演板类型决策规则

本文件只回答一个问题：当前故事与素材更适合哪种导演板类型。

## 1. 连续镜头型 `continuous_shot_board`

适用：

- 一镜到底
- 跟拍
- 推拉摇移连续推进

导演板重点：

- top view 里清楚画出路径
- shot list 体现时间连续
- 角色状态与光线方向不能跳

## 2. 多机位型 `multi_shot_board`

适用：

- 对话反打
- 同一场景多角度覆盖
- 情绪对峙

导演板重点：

- blocking 必须交代轴线与视线方向
- shot list 必须解释切镜理由
- 镜头语言要有节奏递进

## 3. 空间建立型 `spatial_establishing_board`

适用：

- 地点揭示
- 外到内路径
- 复杂空间导航

导演板重点：

- 场景与美术区必须承担更多信息
- top view 要比人物特写更重要
- shot list 从空间建立逐步进入事件发生区

## 4. 动作递进型 `action_progression_board`

适用：

- 打斗
- 奔跑
- 动作拆解

导演板重点：

- shot list 清楚拆出动作起点、中段、高点和收尾
- 镜头语言要服务力量感与节奏
- 灯光与摄影说明要支持动作可读性

## 5. 产品交互型 `product_interaction_board`

适用：

- 广告
- 道具触发
- 产品展示

导演板重点：

- 产品可见性高于花哨构图
- 角色反应与产品交互路径必须明确
- 场景与美术区要为产品提供合理环境

## 6. 转场变化型 `transition_board`

适用：

- 变身
- 状态变化
- 镜像 / 梦境 / 时空切换

导演板重点：

- 必须有中间态
- blocking 要说明变化发生在哪个空间层
- 灯光和质感要帮助转场成立

## 7. 情绪表演型 `emotion_performance_board`

适用：

- 微表演
- 关系反应
- 面部和姿态情绪递进

导演板重点：

- 角色参考区和 shot list 的联动最强
- shot list 重点不是动作，而是反应和拍点
- tone、灯光、摄影说明必须服务表演
