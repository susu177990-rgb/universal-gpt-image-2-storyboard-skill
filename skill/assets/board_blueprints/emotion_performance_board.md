# 情感演绎分镜板蓝图 (统一六区提案表)

*此蓝图将“六区统一分镜”布局应用于侧重于角色内心情感历程的镜头序列，运用面部表情、微动作和视线。*

## 区域 1：头部信息栏 (元数据)
- **标题/场景**: 情感弧线 [插入情感，例如：“意识到背叛”]
- **序列类型**: 情感演绎 (角色研究)
- **帧率 / 运行时长**: 基础 24fps / 目标 5-10 秒
- **氛围/基调**: 私密、脆弱、细腻

## 区域 2：角色与造型锁定栏
- **主体**: 细节化的面部特征（例如：“突出的颧骨，左眉处有轻微疤痕，眼窝深陷”）。
- **化妆/汗水/泪水**: 确立基准状态（例如：“眼眶干燥，哑光肌肤”），以与情感顶点形成对比（例如：“泪水夺眶而出，皮肤泛红”）。
- **服装焦点**: 领口和肩部细节，因为构图将会非常紧凑。

## 区域 3：摄影机与运动规划区
- **机身/镜头**: 人像镜头（例如 85mm 或 100mm 微距），以压缩背景并突出面部特征。浅景深。
- **摄影机运动轨迹**: 
  - 保持摄影机相对稳定。
  - 微量推镜头 (Micro-push-ins): `随着情绪的积聚，极其缓慢、几乎难以察觉地向眼睛推进。`

## 区域 4：表情与动作关键帧区
- 该区域对于此蓝图至关重要。解剖情感的构成：
  1. *眼睛*: 闪烁、凝视、瞳孔放大、热泪盈眶。
  2. *嘴巴*: 轻微的颤抖、咬紧牙关、勉强的笑容破裂。
  3. *肩膀*: 沮丧地下垂或愤怒地紧绷。
  4. *呼吸*: 胸膛剧烈起伏 (深吸气)。

## 区域 5：核心分镜网格 (顺序面板)
*(侧重于微妙性、视线以及渐进式的情感积聚)*

- **面板 1**: 表情起点 (Expression Starting Point)
  - *动作*: 确立角色的基准情感状态。中性或之前的情感。
- **面板 2**: 眼神/视线变化 (Trigger / Gaze Change)
  - *动作*: 展示角色注意到刺激物或对刺激物做出反应。眼睛首先发生变化。
- **面板 3**: 身体微动作 (Body Micro-Movements)
  - *动作*: 微妙的身体反应（肩膀僵硬、轻微倾斜）。
- **面板 4**: 情绪推进 (Emotion Progression)
  - *动作*: 情感在面部肌肉中积聚。前额起皱、下巴收紧。
- **面板 5**: 表情高点 (Expression Peak)
  - *动作*: 情感反应的高潮（例如：充分展现的笑容、眼泪落下、眼睛睁大）。
- **面板 6**: 表情收束 (Expression Resolution)
  - *动作*: 情感的平复或过渡到下一个想法。

## 区域 6：技术规格页脚
- **照明**: 柔和的包围光（例如：大型柔光箱或窗户光）以美化面部，配以明显的眼神光 (catchlight) 让眼睛显得生动。
- **材质/视觉特效**: 皮肤纹理细节（毛孔、细微的汗水），眼睛中的高光 (specular highlights)。
- **色彩脚本**: 肤色冷暖的微妙变化，取决于情感节拍（例如：因恐惧而褪去血色）。
