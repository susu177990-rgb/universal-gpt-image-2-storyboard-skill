# 分镜类型决策规则

本文件只回答一个问题：当前需求最适合哪种 `board_type`。

## 1. 连续镜头型 `continuous_shot_board`

适用：

- 一镜到底
- 跟拍
- 推拉摇移连续推进
- 同一机位逻辑下的完整动作

解决的问题：

- 保持时间连续
- 保持空间连续
- 保持角色走位和屏幕方向连续

必须表现：

- 同一主体的连续运动
- 相对稳定的空间关系
- 摄影机或主体的推进轨迹

避免：

- 突然跳时空
- 同一段动作里突然换身份或换光向

典型逻辑：

起点 -> 运动推进 -> 高点 -> 停点

## 2. 多机位型 `multi_shot_board`

适用：

- 同一场景下需要切角度
- 对话反打
- 同一动作要靠不同景别覆盖

解决的问题：

- 叙事节奏
- 重点切换
- 信息分配

必须表现：

- 明确的切镜理由
- 景别或角度差异
- 视线和 180 度关系

避免：

- 无意义重复切镜
- 破坏人物左右关系

典型逻辑：

建立镜头 -> 主体 A -> 主体 B -> 反应或细节 -> 收束

## 3. 空间建立型 `spatial_establishing_board`

适用：

- 展示地点
- 建立外到内路径
- 说明多个区域之间的关系

解决的问题：

- 让观众搞清楚“这是哪里”
- 让后续动作有空间基础

必须表现：

- 外部或宏观视角
- 入口或阈值
- 内部整体布局
- 核心发生区

避免：

- 还没建立空间就直接怼特写
- 前后布局矛盾

典型逻辑：

外景 -> 入口 -> 内景全貌 -> 核心区域

## 4. 动作递进型 `action_progression_board`

适用：

- 打斗
- 跑跳
- 转身
- 操作设备
- 明确的身体动作分解

解决的问题：

- 把复杂动作拆成模型能理解的节拍

必须表现：

- 起手
- 蓄力
- 中段
- 高点
- 收尾

避免：

- 跳过动作高点
- 机位乱变导致看不清动作关系

典型逻辑：

起始姿态 -> 蓄力 -> 动作中段 -> 高点 -> 结束

## 5. 产品交互型 `product_interaction_board`

适用：

- 广告
- 道具触发
- 手和物体的交互
- 强调产品识别度的镜头

解决的问题：

- 防止产品被忽略、挡住或变形

必须表现：

- 产品位置
- 角色与产品距离
- 接触点
- 使用状态
- 最终展示状态

避免：

- 手把产品完全挡住
- 各格产品长得不一样

典型逻辑：

发现 -> 接近 -> 接触 -> 特征展示 -> 最终展示

## 6. 转场变化型 `transition_board`

适用：

- 变身
- 换装
- 时空切换
- 状态 A 到状态 B 的过渡

解决的问题：

- 把抽象变化变成可见中间态

必须表现：

- 前状态
- 触发点
- 中间态 1
- 中间态 2
- 后状态
- 稳定尾帧

避免：

- 从 A 直接跳到 B
- 触发机制不明确

典型逻辑：

前状态 -> 触发 -> 中间态 1 -> 中间态 2 -> 后状态 -> 稳定尾帧

## 7. 情绪表演型 `emotion_performance_board`

适用：

- 面部特写
- 微表演
- 情绪递进
- 角色反应

解决的问题：

- 防止只拍动作，不拍表演

必须表现：

- 起始表情
- 触发时刻
- 微动作
- 情绪积累
- 表情高点
- 收束

避免：

- 空泛标签式情绪
- 没有表情证据的“开心”“伤心”

典型逻辑：

中性 -> 注意到 -> 微反应 -> 情绪推进 -> 高点 -> 收束
