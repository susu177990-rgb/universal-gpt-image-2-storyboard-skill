# 动作推进分镜板蓝图 (统一六区提案表)

*此蓝图将“六区统一分镜”布局应用于旨在将复杂的身体动作（例如：格斗、舞蹈、设备操作）拆解为清晰、可生成的步骤的镜头序列。*

## 区域 1：头部信息栏 (元数据)
- **标题/场景**: 动作拆解 [插入具体动作，例如：“回旋踢”]
- **序列类型**: 动作推进
- **帧率 / 运行时长**: 基础 24fps / 目标 3-5 秒
- **氛围/基调**: 动感、充满冲击力、物理真实感

## 区域 2：角色与造型锁定栏
- **主体**: 描述大量参与动作的身体特征（例如：“穿着轻量级装备的格斗家，手上缠着绷带，运动型身材”）。
- **服装动态**: 服装如何随着运动产生反应（例如：“宽松的面料在空气中甩动”）。
- **道具锁定**: 使用的武器或工具（例如：“碳纤维齐眉棍”）。提供参考元素以保持比例。

## 区域 3：摄影机与运动规划区
- **机身/镜头**: 高速运动摄影机（例如 RED V-Raptor），调整快门开角以获取动态模糊。大光圈快速镜头（例如 35mm f/1.4）。
- **摄影机运动轨迹**: 
  - 跟随重心。
  - 例如：`向左摇摄跟随击打动作 -> 撞击瞬间急推 (Crash zoom) -> 倾斜镜头 (Dutch angle) 强调力量感。`

## 区域 4：表情与动作关键帧区
- 包含 3-4 个关键生物力学细节的局部特写：
  1. *足部轴心*: 脚踝扭动踩实地面。
  2. *肌肉张力*: 前臂或肩部紧绷蓄力。
  3. *面部发力*: 咬紧牙关，视线紧盯目标。
  4. *撞击点*: 指关节与表面接触的瞬间。

## 区域 5：核心分镜网格 (顺序面板)
*(侧重于节奏，避免跳过动作高点，保持清晰的方向性)*

- **面板 1**: 起始姿态 (Starting Pose)
  - *动作*: 确立主体在动作开始前的基准状态。
  - *摄影机*: 中全景 (Medium Full Shot)。
- **面板 2**: 动作准备 (Preparation / Anticipation)
  - *动作*: 展示蓄力或意图（例如：将拳头向后拉、下蹲）。
- **面板 3**: 动作中段 (Mid-Action)
  - *动作*: 捕捉进行中的运动。高速动态模糊。
- **面板 4**: 动作高点 (Action Peak)
  - *动作*: 运动的顶点（例如：拳头击中的瞬间、跳跃的最高点）。必须完美构图。
- **面板 5**: 动作结束 (Ending / Follow-through)
  - *动作*: 展示动作的物理收尾和缓冲恢复。
  - *摄影机*: 更宽的景别以展示动作余波。

## 区域 6：技术规格页脚
- **照明**: 强硬的方向光以雕塑肌肉线条；边缘光 (Edge lighting) 将快速移动的肢体与背景分离开来。
- **材质/视觉特效**: 地板扬起的灰尘、额头飞溅的汗水、轻微的速度线大气扭曲效果。
- **色彩脚本**: 降低背景饱和度，并在主体上点缀高对比度色彩以引导视线。
