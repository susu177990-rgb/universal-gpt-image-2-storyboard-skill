# 示例 03：动作递进请求（重拳出击）

## 输入

- 输入模式：素材与文本混合
- 输出目的：给人审核 / 提案
- 主动作：角色朝镜头挥出重拳
- 素材：
  - `fighter_ref`：角色

## 中间结果重点

### `storyboard_plan`

- 分镜类型：`action_progression_board`
- 格数：4
- 每格功能：
  - 第 1 格：起始姿态
  - 第 2 格：蓄力
  - 第 3 格：爆发
  - 第 4 格：冲击成立

## 预期输出

### Markdown 提示词片段

````md
# 重拳出击

- 连续性重点：角色身份不变，拳头推进方向稳定
````
